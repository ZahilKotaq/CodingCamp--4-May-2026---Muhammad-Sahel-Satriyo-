# To-Do List Life Dashboard

Sebuah aplikasi web *dashboard* produktivitas pribadi yang dibangun sebagai proyek akhir untuk tugas Coding Camp. Aplikasi ini membantu pengguna untuk mengatur hari mereka dengan lebih baik melalui fitur manajemen waktu dan tugas.

Fitur:
1. Sapaan & Jam Digital (Greeting): Menampilkan waktu secara real-time dan ucapan yang berubah otomatis sesuai waktu (Pagi/Siang/Sore/Malam).
2. Pengatur Waktu Fokus (Focus Timer): Timer Pomodoro yang bisa di-start, pause, dan di-reset, lengkap dengan suara notifikasi saat waktu habis.
3. Daftar Tugas (To-Do List): Pengguna dapat menambah, mencoret (selesai), dan menghapus tugas. Semua tugas otomatis tersimpan di perangkat pengguna.
4. Tautan Cepat (Quick Links): Fitur untuk menyimpan bookmark atau tautan situs web favorit pengguna secara lokal.

Fitur Tambahan 
Proyek ini juga dilengkapi dengan beberapa fitur ekstra untuk meningkatkan kenyamanan pengguna:
Mode Terang / Gelap (Light & Dark Mode): Desain yang bisa diganti sesuai kenyamanan mata pengguna.
Mencegah Duplikat Tugas: Sistem akan memperingatkan pengguna jika mereka memasukkan nama tugas yang sama persis.
Filter & Urutkan Tugas: Pengguna bisa menyaring tampilan daftar tugas berdasarkan status (Semua / Belum Selesai / Selesai).
Waktu Pomodoro: Menyediakan tombol cepat untuk memilih waktu fokus (25 menit, 10 menit, 5 menit, atau 50 menit).
